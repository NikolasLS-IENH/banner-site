# Banner-Site

Repositório do trabalho de desenvolvimento web para recriar a seção de destaque do site [eduardavelho.com](https://eduardavelho.com/), usando HTML e CSS.

Cada grupo da turma ficou com uma parte do site. Aqui está apenas a seção “Em destaque” (banner inicial).

## Estrutura

- `index.html` – Estrutura principal da página.
- `css/style.css` – Estilos gerais.
- `css/banner.css` – Estilos do banner / destaque.
- `imagens/` – Logo, avatar e imagem de fundo.

## Seção “Em destaque”

A seção `#banner` contém:

- Fundo em degradê (rosa para amarelo).
- Card esquerdo com:
  - Título “Em destaque”.
  - Texto da postagem.
  - Botão “Acessar postagem”.
  - Imagem de fundo aplicada no CSS.
- Card direito com:
  - Tags de área (JavaScript, Visualização de dados, etc.).
  - Foto de perfil.
  - Nome e subtítulo.
  - Descrição.
  - Links para redes sociais.

O layout é responsivo, ajustando os cards para telas menores.

## Como abrir o projeto

1. Clone o repositório:

   ```bash
   git clone https://github.com/NOMEDOUSUARIO/Banner-Site.git
   ```

2. Abra a pasta no editor.
3. Abra o arquivo `index.html` no navegador.
